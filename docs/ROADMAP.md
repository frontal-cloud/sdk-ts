# Project Roadmap

This document outlines the planned work and future goals for the SDK Monorepo.

## Phase 1: Foundation (Current)
- [x] Initial monorepo setup with Bun and Turborepo.
- [x] Core packages initialization: AI, Compute, Functions, Storage.
- [x] Standardized CI/CD workflow and release process.

## Phase 2: Core Enhancements (Q1 2026)
- [ ] **AI**: Add support for more LLM providers (Anthropic, Google).
- [ ] **Compute**: Implement localized runners for edge environments.
- [ ] **Functions**: Improve cold start times and add more triggers.
- [ ] **Storage**: Support more cloud providers (Azure, Google Cloud).

## Phase 3: Developer Experience (Q2 2026)
- [ ] **CLI**: Create a dedicated CLI for managing SDK projects.
- [ ] **Documentation**: Inter-active documentation and playground.
- [ ] **Examples**: Comprehensive suite of example projects.

## Phase 4: Scaling & Security (Q3 2026)
- [ ] **Observability**: Built-in tracing and monitoring.
- [ ] **Security**: Enhanced encryption at rest and in transit.
- [ ] **Performance**: Benchmarking suite and optimizations.

---

*Note: This roadmap is subject to change based on community feedback and priorities.*
