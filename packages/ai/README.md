# @frontal-cloud/ai

AI integration and utilities for the SDK.

## Installation

```bash
bun add @frontal-cloud/ai
```

## Features

- **Model Integration**: Easy access to LLM providers.
- **Prompt Templates**: Manage your prompts effectively.
- **Streaming**: Native support for streaming responses.

## Usage

```typescript
import { ai } from "@frontal-cloud/ai";

console.log(ai());
```
