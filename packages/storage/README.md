# @frontal-cloud/storage

Scalable storage interactions for the SDK.

## Installation

```bash
bun add @frontal-cloud/storage
```

## Features

- **Unified Interface**: Interact with different storage providers using a single API.
- **High Performance**: Optimized for large data transfers.
- **Security**: Built-in support for encrypted storage.

## Usage

```typescript
import { Storage } from "@frontal-cloud/storage";

const storage = new Storage();
// ... use storage
```
