# @frontal-cloud/logging

Scalable logging interactions for the SDK.

## Installation

```bash
bun add @frontal-cloud/logging
```

## Features

- **Unified Interface**: Interact with different logger providers using a single API.
- **High Performance**: Optimized for large data transfers.
- **Security**: Built-in support for encrypted logs.

## Usage

```typescript
import { Logging } from "@frontal-cloud/logging";

const logger = new Logging();
// ... use logger
```
