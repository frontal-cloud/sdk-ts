# @frontal-cloud/functions

Serverless functions orchestration layer.

## Installation

```bash
bun add @frontal-cloud/functions
```

## Features

- **Function Deployment**: Deploy functions to various environments.
- **Triggers**: Trigger functions via HTTP, events, or schedules.
- **Orchestration**: Manage complex function workflows.

## Usage

```typescript
import { Functions } from "@frontal-cloud/functions";

const functions = new Functions();
// ... use functions
```
