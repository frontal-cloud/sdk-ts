# Security Policy

## Supported Versions

Currently, we only support the latest version of the SDK. If you are using an older version, please update to the latest release to ensure you have the latest security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Yes             |
| < 1.0   | ❌ No              |

## Reporting a Vulnerability

We take security seriously and appreciate your help in making this project more secure.

If you find a security vulnerability, please **do not open a public issue**. Instead, please report it privately to the maintainers at [security@example.com](mailto:security@example.com).

When reporting a vulnerability, please include:
- A description of the vulnerability.
- Steps to reproduce (ideally with a small code snippet).
- Potential impact.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible. We will also coordinate a public disclosure once a fix is available and has been tested.
